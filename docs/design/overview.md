## Region System (Front of Card)

Every non-Journey card belongs to one of Uganda's four regions. The region is displayed at the **top-left corner** of the front as a **single bold letter inside a colored circle**.

| Region | Initial | Color | Hex |
|---|---|---|---|
| Central | C | Green | #2D6A4F |
| Western | W | Blue | #1B4F8A |
| Northern | N | Yellow/Gold | #E5A020 |
| Eastern | E | Red | #C0392B |
| Universal (Journey) | U | Silver/White | #8E9BAE |

Journey cards use **U** (Universal) and a neutral silver — they belong to no region and can be played anywhere.

## Front Layout System

All cards share the same front layout structure:

```
┌─────────────────────────────┐
│ [Region Badge]  [Card Type] │  ← top bar
│                             │
│                             │
│      FULL BLEED             │
│      ILLUSTRATION           │
│                             │
│                             │
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│  ← dark gradient overlay
│  CARD NAME                  │
│  Region Name                │
└─────────────────────────────┘
```

- **Top left** — Region badge (colored circle + bold single letter)
- **Top right** — Card type label (small uppercase, e.g. "Destination", "Community")
- **Center** — Full bleed illustration, painterly style
- **Bottom** — Dark gradient overlay with card name in bold serif font, region name below in smaller text

## Back Layout System

All backs are **symmetrical and decorative** — no card-specific information. They look like premium playing card backs.

```
┌─────────────────────────────┐
│  ╔═══════════════════════╗  │
│  ║                       ║  │
│  ║   [decorative         ║  │
│  ║    pattern/border]    ║  │
│  ║                       ║  │
│  ║    🦅 (crane icon)    ║  │
│  ║  DISCOVER UGANDA      ║  │
│  ║                       ║  │
│  ║   [decorative         ║  │
│  ║    pattern/border]    ║  │
│  ║                       ║  │
│  ╚═══════════════════════╝  │
│   Travel Like a Ugandan     │
└─────────────────────────────┘
```

- **Center** — Grey Crowned Crane icon (Uganda's national bird) + "Discover Uganda" in serif font
- **Border/Pattern** — African-inspired geometric or botanical pattern, different color per card type
- **Bottom micro text** — "Travel Like a Ugandan"

---

## Illustration Style

All card illustrations follow the same art direction:

- **Style** — Painterly editorial illustration (NOT photorealistic, NOT cartoon)
- **Quality reference** — Wingspan, 7 Wonders, Ticket to Ride card art
- **Lighting** — Warm, golden hour where possible
- **Color palette** — Rich, warm African tones: deep greens, amber, ochre, earthy brown, ivory
- **Feel** — Adventurous, warm, cultural — National Geographic meets premium board game

---

## QR Code (Back of Card)

Select cards (primarily Destination cards) include a QR code on the **front bottom-right corner**, not the back.

- Placed inside a small rounded decorative frame
- Styled in the card's region color (not plain black)
- Labeled: **"Scan to Unlock"** in micro text below
- Minimum size: 2cm × 2cm (still scannable)
- The QR does not disrupt the illustration — it sits in the gradient overlay area at the bottom
